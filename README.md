**Project structure**
```
src/
├── assets/
├── components/
│   ├── AppHeader.vue
│   ├── CartAddRemove.vue
│   ├── CartBTN.vue
├── layouts/

├── store/

│   ├── ProductList.vue

└── main.js
```

**Used components**

- VueX
- Axios
- Vue-router
- Bootstrap

**Install**

1- Download the repository
2- cd shooping-cart
3- npm install
4- npm run dev
